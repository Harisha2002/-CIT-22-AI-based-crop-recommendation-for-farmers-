\# 🌾 AI-Based Crop Recommendation for Farmers



A smart web application that recommends the most suitable crops 

based on soil and weather data using AI/ML techniques.



\## 📌 About the Project



This project helps farmers make better decisions by analyzing 

soil nutrients and weather conditions to suggest the best crops 

to grow. Built as a college project under CIT-22.



\## 🚀 Features



\- 🌱 Crop recommendation based on soil data (N, P, K, pH)

\- 🌤️ Weather-based analysis

\- 📊 Interactive dashboard for farmers

\- 🔐 User authentication via Supabase



\## 🛠️ Tech Stack



| Technology | Usage |

|------------|-------|

| React | Frontend UI |

| Supabase | Backend \& Database |

| Vite | Build Tool |
